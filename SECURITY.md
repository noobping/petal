# Security Policy

## Scope

This repository contains a Rust client for listen.moe.

I do **not** own or operate the listen.moe radio service, servers,
or backend infrastructure.

- Issues affecting the [listen.moe](https://listen.moe/) radio service itself should be
  reported via the [official listen.moe Discord](https://listen.moe/discord).
- Issues specific to this Rust client fall within the scope of this project.

## Supported Versions

This is a hobby project, provided "as is" under the MIT license.
Security updates are not guaranteed.

| Version | Supported |
| ------- | --------- |
| Latest  | ✔️        |
| Older   | ❌        |

## Reporting a Vulnerability

This client streams public radio content and does not handle
personal data, authentication, or secrets.

Security issues are handled publicly.  
You may open a GitHub issue or pull request to report a problem.

There is no formal response timeline, but reports and contributions
are appreciated.
